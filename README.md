# 基于 Spring Boot 的中小学教育平台
一个面向中小学的在线教育平台，采用前后端不分离架构（Vue 编译后的静态文件直接由 Spring Boot 托管）。平台分为**管理后台**与**用户门户**两端，覆盖课程教学、在线考试、作业管理、资料共享与交流互动等核心场景，支持管理员、教师、学生三种角色。

## 技术栈
| 类别 | 技术 |
| --- | --- |
| 后端框架 | Spring Boot 2.7.18 |
| 持久层 | MyBatis-Plus |
| 数据库 | MySQL 8.0 |
| 缓存 | Redis |
| 前端 | Vue（编译后静态部署于 `resources` 下） |
| 其他 | Baidu AI SDK、Fastjson、Hutool、Apache POI |
| 运行环境 | JDK 17、Maven 3.9 |

## 角色说明
- **管理员**：平台的最高管理者，管理教师、学生、课程及各类基础数据。
- **教师**：发布课程与资料、布置和批改作业、组织在线考试。
- **学生**：报名课程、学习资料、提交作业、参加考试、参与讨论。

## 主要功能
### 用户与权限
基于 Token 的登录鉴权，区分管理员、教师、学生三种角色，不同角色登录后看到不同的功能菜单与数据权限。

### 课程管理
- **课程信息**：教师发布课程，包含课程名称、类型、封面、上课时间地点、课程时长与详情。
- **课程类型**：对课程按学科分类（数学、语文、英语等）。
- **课程报名**：学生在线浏览并报名感兴趣的课程。

### 教学资料
- **课程资料**：教师上传课件、教案、习题、试卷等教学资源，支持图片、视频、附件下载。
- **资料类型**：对资料进行分类管理。

### 作业管理
- **作业信息**：教师针对课程布置作业，设置作业要求与提交时间。
- **作业提交**：学生在线提交作业，教师进行批改，记录批改状态。

### 在线考试
- **试题库 / 试题**：维护题库与单题，支持组卷。
- **在线考试 / 试卷**：教师组织考试，学生在线作答。
- **考试记录 / 成绩信息**：自动记录考试结果，统计与查询学生成绩。

### 交流互动
- **在线讨论**：平台论坛，师生发帖交流。
- **课程 / 资料评论**：针对具体课程或资料进行评论互动。
- **收藏**：学生收藏感兴趣的课程与资料，便于后续查看。

### 内容与系统管理
- **公告信息**：发布平台公告，支持公告分类。
- **关于我们 / 系统简介**：平台介绍与基本信息展示。
- **系统配置**：轮播图等基础配置项管理。

## 运行说明
### 环境准备
1. 安装 JDK 17、Maven 3.9、MySQL 8.0、Redis，并确保 MySQL 与 Redis 已启动。
2. 创建数据库 `wyp_springboot`，导入项目根目录下的 SQL 脚本初始化数据。

### 配置
数据库连接配置位于 `src/main/resources/application.yml`，默认连接本地 MySQL（用户名 `root`，密码为空），可按实际环境修改。

### 启动
通过 IDEA 运行启动类 `SpringbootSchemaApplication`，或使用 Maven 打包后运行。服务默认监听 **8080** 端口。

### 访问地址
- 管理后台：`http://localhost:8080/springboot6056cc19/admin/index.html`
- 用户门户：`http://localhost:8080/springboot6056cc19/front/index.html`
